# 🚆 Railway Info Web App

Proyek ini adalah sebuah website donasi online untuk masjid yang membutuhkan, dibangun menggunakan **HTML**, **PHP**, dan **Tailwind CSS**. Tujuannya adalah menyajikan antarmuka yang responsif dan mudah digunakan untuk pengguna yang ingin memberikan donasinya dan pengguna dapat membuka donasi untuk masjid.

---

## 🔧 Teknologi yang Digunakan

- HTML5
- PHP 8
- Tailwind CSS

---

## 🖥️ Cara Menjalankan

1. Pastikan kamu punya server PHP (bisa pakai Laragon).
2. Upload seluruh isi project ke folder `www/` (jika lokal).
3. Buat database dengan nama `sistem_donasi_masjid`.
4. Import file yang ada di folder `db`.
5. Akses halaman utama di `http://localhost/pedulimasjid/home`.

---

## 💡 Fitur

- Navigasi sederhana dan cepat
- Siap dikembangkan lebih lanjut (misalnya integrasi database)
